		<category name="Mashers">
			<category name="Description">
				<comment>By Aaron0000</comment>
				<comment>Changes the double accessory of all pistols into the (in)famous Masher accessory.</comment>
				<comment>Mashers have a minor penalty in critical damage.</comment>
				<comment>Hyperion mashers only have x5 projectiles. They are however more accurate.</comment>
				<comment>Maliwan and Torgue mashers only have x6 projectiles, due to already having splash.</comment>
				<comment>Dahl mashers have a longer delay between series of bursts.</comment>
				<comment>Maggie's masher is different than normal mashers. Due to how the double accessory was modified, the Maggie's damage wound up being tripled, so damage was chopped by 66% to return it to normal. Then the pellet count was quadrupled to achieve the same overall damage boost as other mashers.</comment>
				<comment>Unkempt Harold Mashers only have 5 projectiles instead of 7 to keep the damage roughly in line with the standard Double Penetrating Unkempt Harold.</comment>
				<comment>Comes with custom prefixes: Jakobs = Mashin', Hyperion = Teamplay, Torgue = Septuple Penetrating, Bandit = buncha, Maliwan = Septenary, Dahl = Multitude, Tediore = Seven for One, Vladof = Bolshevik.</comment>
				<comment>NOTES:</comment>
				<comment>If using the 88 Fragnum replacer for the Unkempt Harold, there is a patch for it down below to make it function correctly as a Masher.</comment>
				<comment>If using the Community Patch, there is a fix for the Wanderlust to make it mash properly. Located at the bottom.</comment>
				<comment>If using the Revolvers mod and you want to use this mod's version of mashers (that "mashes" all pistols, instead of just Jakobs/Maliwan/Torgue), just activate this mod after Revolvers.</comment>
				<comment>If using the Revolvers (Jakobs-Barrel version) mod, there is a fix for Maggie and Unkempt Harold each to make them mash properly. Located at the bottom.</comment>
				<comment>If using the Violator replacer for the Pocket Rocket, there is a patch for it down below to make it function correctly as a Masher.</comment>
				<comment>If using Atlas Reborn, be sure to re-activate the Infinity masher fix line. That, or activate mashers after Atlas Reborn.</comment>
				<comment>If using Infinity Beyond, be sure to re-activate the Infinity masher fix line. That, or activate mashers after Infinity Beyond.</comment>
